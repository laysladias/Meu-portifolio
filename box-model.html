Box Model

Definição do Box Model

Cada elemento HTML é representado como uma caixa retangular, conhecida como "box". O modelo de caixa consiste em: conteúdo, preenchimento, borda e margem. Abaixo está um exemplo visual de cada uma dessas propriedades.

Conteúdo

Conteúdo: O que está dentro da caixa (texto, imagem, etc).

Preenchimento (Padding): Espaço entre o conteúdo e a borda.

Borda: Contorno ao redor do conteúdo e preenchimento.

Margem: Espaço fora da borda, separando o elemento de outros ao redor.
