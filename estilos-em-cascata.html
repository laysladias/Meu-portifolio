Estilos Em Cascata

Cascata

O conceito de "cascata" em CSS refere-se à forma como o navegador decide qual regra aplicar quando há várias regras para um mesmo elemento.

Título Principal(Id)

IDs são usados para estilizar elementos únicos. Cada ID deve ser único dentro da página definido com # para aplicar estilos específicos via CSS na página.

Classe

Um elemento pode ter várias classes. As classes são definidas com um ponto (.) no CSS.

Elemento

É todos os elementos do mesmo tipo como (p,h1...) na página.
